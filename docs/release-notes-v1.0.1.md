## Release Notes

- Tooltip now lists up to 4 running job names, then shows `+N more`
- Keeps global running execution count in navbar badge
- Maintains cross-project running detection (`all readable projects`)

## Installation

1. Download `ui-active-jobs-navbar-indicator-1.0.1.jar` from this release
2. In Rundeck: System Menu -> Plugins -> Upload Plugin
3. Restart Rundeck if required by your deployment

## Compatibility

- Rundeck Community: 5.x
- Runbook Automation (Self-Hosted): 5.x
