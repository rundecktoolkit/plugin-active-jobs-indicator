## Release Notes

- Adds UI plugin `ui-active-jobs-navbar-indicator`
- Shows active execution spinner and count in top navbar
- Removes debug text from UI rendering

## Installation

1. Download `ui-active-jobs-navbar-indicator-1.0.0.jar` from this release
2. In Rundeck: System Menu -> Plugins -> Upload Plugin
3. Restart Rundeck if required by your deployment

## Compatibility

- Rundeck Community: 5.x
- Runbook Automation (Self-Hosted): 5.x
